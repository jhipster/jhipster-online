Generated applications can be deployed on Azure Container Apps in a few easy steps, please refer to the readme file in the generated project for the deployment guide.

Detailed documentation is also available here.

Provision your Azure Resources using NubesGen
https://nubesgen.com

Please note that the current version of this form only supports the following selections within NubesGen:

  • Application Type: Spring Boot Application
  • Database: Azure Database for PostgreSQL and Azure Cosmos DB

Services, Databases and selections not listed above are currently not supported and code will not be generated by this form if provisioned within NubesGen.

More details

Maven will be the build tool used for the project.
React will be the default front-end framework used for the project.
PostgreSQL will default to passwordless database connections as it's more secure option.
The blueprint generator-jhipster-azure-container-apps will be added.
